"Documentacion del proyecto"

DOCUMENTACIÓN DEL PROYECTO WEB Autobiografía en HTML 1. Introducción El presente proyecto consiste en el desarrollo de un sitio web estático utilizando HTML básico, cuyo propósito es presentar una autobiografía digital de manera organizada, clara y accesible. A través de diferentes páginas interconectadas, se expone información personal, experiencias de vida y aspectos relevantes del desarrollo académico y personal del autor. Este trabajo permite aplicar los conocimientos fundamentales de desarrollo web, especialmente en la estructuración de documentos HTML y la organización de contenido. 2. Objetivo del Proyecto El objetivo principal es diseñar un sitio web funcional que permita mostrar una autobiografía completa, haciendo uso exclusivo de HTML. Se busca: Organizar información personal en diferentes secciones Facilitar la navegación entre páginas Aplicar correctamente etiquetas HTML Presentar contenido de forma clara y estructurada 3. Estructura del Sitio Web El sitio está compuesto por múltiples páginas HTML, cada una dedicada a una sección específica: Inicio: Presentación general del sitio Información Personal: Datos básicos del autor Historia Familiar: Contexto familiar Niñez: Experiencias durante la infancia Adolescencia: Etapa de crecimiento personal Formación: Trayectoria académica Intereses: Áreas de interés personal Metas: Objetivos a futuro Gustos: Preferencias personales Multimedia: Imágenes relacionadas con la autobiografía Todas las páginas están conectadas mediante un menú de navegación común. 4. Descripción Técnica del Código 4.1 Estructura básica HTML Cada archivo HTML sigue una estructura estándar que incluye: Declaración del tipo de documento () Etiqueta raíz con idioma definido Sección con metadatos Sección con el contenido visible 4.2 Encabezado (Header) El encabezado contiene el título principal del sitio y una breve descripción. Este elemento se repite en todas las páginas para mantener consistencia. 4.3 Navegación (Nav) El menú de navegación permite acceder a todas las secciones del sitio. Está construido mediante una tabla que organiza los enlaces de forma horizontal. Esto facilita la experiencia del usuario al moverse entre las diferentes páginas. 4.4 Contenido Principal (Main) En esta sección se presenta la información específica de cada página, utilizando: Párrafos para texto descriptivo Listas ordenadas y desordenadas Tablas para datos estructurados Imágenes para contenido visual 4.5 Sección Multimedia La página de multimedia incluye imágenes organizadas mediante las etiquetas
